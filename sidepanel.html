Wiki Markup Syntax

  • **text** or '''text''' - Bold
  • //text// or ''text'' - Italic
  • __text__ - Underline
  • ~~text~~ - Strikethrough
  • [[link text]] - Internal link
  • [[url|label]] - External link
  • * item - Bullet list
  • # item - Numbered list
  • == Heading == - H2 heading
  • === Heading === - H3 heading
  • ---- - Horizontal line
  • `code` - Inline code

Features

  • Highlight text: Select text on any page, right-click → "Highlight Only"
  • Add to notes: Select text → right-click → "Add as Heading" or "Add as Paragraph"
  • Remove highlight: Right-click on highlighted text → "Remove Highlight" (or Alt/Ctrl + Click)
  • Per-page notes: Notes are saved separately for each webpage
  • Export: Click Export or press Ctrl+E to save notes and highlights
  • Keyboard shortcuts: Ctrl+S (save), Ctrl+P (preview), Ctrl+E (export)
